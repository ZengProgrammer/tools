# Tools

A cross-platform desktop application built with Tauri v2 and Vue 3.

## Tech Stack

- **Frontend**: Vue 3 + TypeScript + Element Plus + Vite
- **Backend**: Tauri v2 (Rust)
- **Package Manager**: pnpm

## Getting Started

```bash
pnpm install
pnpm tauri dev
```

## Build

```bash
pnpm tauri build
```
