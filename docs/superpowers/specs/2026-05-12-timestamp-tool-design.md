# 时间戳转换工具 — 设计

## 目标

新增时间戳转换工具，纯前端功能。输入 Unix 秒级时间戳转换为日期时间格式 `YYYY-MM-DD HH:mm:ss`，或输入日期时间反向转换为秒/毫秒/微秒/纳秒级时间戳。

## 交互规则

| 操作 | 行为 |
|------|------|
| 页面打开 | 左侧秒级时间戳每秒 +1，右侧静态显示对应日期 + 四种精度时间戳 |
| 修改任一侧输入 | 刷新停止，时间戳不动，两侧都不变 |
| 点击「转换」 | 根据有输入的一侧更新另一侧 |
| 无效输入 | 秒级时间戳必须是 10 位数字；日期须为合法日期；显示红色提示 |
| 点击「🔄 刷新」 | 清空输入，恢复左侧每秒 +1 + 右侧同步 |

## 技术方案

- 纯前端，无后端依赖
- 使用 JavaScript `Date` 和 `getTime()` 进行双向转换
- 秒级时间戳 = `Math.floor(date.getTime() / 1000)`
- 毫秒 = `date.getTime()`
- 微秒 = `date.getTime() * 1000`
- 纳秒 = `date.getTime() * 1000000`
- 秒级自动流逝用 `setInterval(1000)`
- 时区默认 `Asia/Shanghai`，用 `Intl.DateTimeFormat('zh-CN', { timeZone: 'Asia/Shanghai' })` 格式化
